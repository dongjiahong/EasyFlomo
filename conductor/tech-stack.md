# Technology Stack

## Core Technologies
- **Programming Language:** TypeScript
- **Frontend Framework:** React
- **Build Tool:** Vite

## Key Libraries
- **Icons:** lucide-react
- **Markdown Rendering:** react-markdown
- **Visualization:** react-activity-calendar
- **UI Components:** react-tooltip

## Data & Storage
- **Local Storage:** IndexedDB (for offline capability)
- **Synchronization:** WebDAV (for cross-device sync and user ownership)

## AI & Intelligence
- **AI Model:** Gemini API (for tagging, summaries, and insights)
